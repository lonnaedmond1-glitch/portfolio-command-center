The Challenge

Operational data across the organization was fragmented into static spreadsheets, causing a 3-day delay in billing, margin bleed on at-risk projects, and high compliance risk for fleet drivers.

The System Solution

I architected this dark-mode, high-density dashboard to centralize operations. By tying in real-time webhook data and an LLM logic layer, the system proactively detects conflicts instead of just displaying metrics.

AI Diagnostic Layer

Notice the Solve 🪄 buttons in the Portfolio. They ping an underlying inference API (mocked here) to ingest 5 sources of field data and output a 2-sentence resolution plan for the PM.
